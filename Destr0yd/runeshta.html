Virus.Win32.Neshta Analysis

History

Neshta was written in Belarus in Delphi (according to Wikipedia) in 2005. The virus is compatible only with Windows OS . Interestingly, the author has never been found...
Also, it has different names in antivirus databases:
Virus.Win32.Neshta.a - Kaspersky Anti-Virus
Win32.HLLP.Neshta - Dr. Web
Win32.Neshta - NOD32
Win32.Neshuta - Symantec (yes, Neshuta, that's what Symantec wrote in their databases)
Win32.Apanas - Avast
Virus.Win32.Neshta.B - 360 Total Security

Description

First, the virus searches for files with the .exe extension on your computer, infects them, and creates a copy of itself in the Windows folder. The virus changes the registry parameter (HKCR\exefile\shell\open\command) to @="%WINDIR%\svchost.com \"%1\" %*". After this change, every executable file will launch NestA. This will allow NestA to spread more quickly.

NestA does not have any destructive or graphical payloads.
