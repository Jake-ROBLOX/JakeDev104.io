How does ChilledWindows.exe Joke Virus Work?

ChilledWindows.exe is a popular Joke Program made by GAMELASTER in 2016. It was designed to "prank" the user into thinking something was up with their PC by rotating the screen randomly and showing random error messages. It does not actually cause any damage to the users PC, hence it is called a "Joke Program".

In this article, I aim to take a deep dive into the ChilledWindows.exe program and explore how it works.

Warning

Please Take Caution when running ANY Malware Samples, whether they are jokes or not.

Launching The Program

Upon launching ChilledWindows.exe, the screen flashes a solid white colour, this is where the program takes a screenshot of your entire screen.

The screenshot then flips multiple times back and forth, then it rotates so the screenshot aligns with the top right corner and the bottom left corner.

It then "Alt+Tabs" into a different window, which displayed a Blue Screen of Death (BSOD) background with tons of errors, which contain what appears to be Japanese text. The errors are synced with the background music.

After the song ends, the program exits. If you relaunch the program and press alt+f4 to exit the program prematurely, you will find a file on your destop called "chilledwindows.mp4". Thats right, this whole time the program is just running a video and replacing parts of it with the screenshots of your desktop. If you double click on the video, you can see which parts contain the screenshot of your desktop.

Final Conclusions

The program does not alter computer files, the only suspicious thing that it does it drops the video file, which isnt harmful. A brilliant YouTuber known as Enderman has also analysed this malware. You can find his video on the same subject here.
